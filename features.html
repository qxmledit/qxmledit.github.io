Features

discover how QXmlEdit can help you
  • Split Files - easily split a giant XML file in smaller fragments or apply a a filter saving the results in a set of folders.

    Tutorial

  • Advanced Search - Use a set of options to search in the XML components, even in base64 coded text, and reformat the tree to highlight the results. Use XPath syntax for adavanced queries.

  • Customizable View - the editor can show you the XML tree using different view modes. You can filter the display of attributes or get information about the number of childen.

  • Compare XML - compare XML entities, elements, attributes and so on and view the results in graphical form or get a list of differences.

  • Compare XSD - graphically compare XML Schema files viewing differences.

  • View XSD - view XSD as a set of zoomable objects in a graphic panel.

  • Anonymize data - clean XML from sensible data before using them in development.

  • Sessions - let QXmlEdit collect your file accesses for a work session you define, so you can recall your resources in a glimpse.

  • View Structure - explore the structure of a XML file so big you can't edit it. See how the data are displaced.

  • Edit XSD types - edit XML Schema files with specialized dialogs.
